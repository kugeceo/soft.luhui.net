除了微信号，还可以使用微信链接添加哪些内容？

除了通过微信号添加好友外，微信协议链接还支持多种功能，主要包括以下几类：

1. **添加微信群**
通过群聊的邀请链接（通常是临时有效），格式类似：
```html
<a href="weixin://dl/chat?invitecode=群邀请码">加入微信群</a>
```
（注：群邀请码由群管理员生成，有有效期限制）

2. **打开微信小程序**
通过小程序的原始ID打开指定小程序：
```html
<a href="weixin://dl/business/?t=小程序原始ID">打开小程序</a>
```
（小程序原始ID格式如：gh_xxxxxxxxxxxx）

3. **打开微信公众号**
通过公众号的ID打开公众号主页：
```html
<a href="weixin://dl/officialaccount?公众号ID">关注公众号</a>
```

4. **发起微信支付**
通过微信支付的参数唤起支付界面（需要商户平台配置）：
```html
<a href="weixin://wxpay/bizpayurl?pr=支付参数">微信支付</a>
```

5. **分享内容到微信**
唤起微信分享界面（需配合具体内容参数）：
```html
<a href="weixin://dl/share?分享参数">分享到微信</a>
```

需要注意的是，这些链接大多依赖微信的私有协议，其可用性可能受微信版本、系统平台（iOS/Android）以及微信官方政策调整的影响。部分功能（如支付、小程序）需要提前在微信开放平台完成开发者认证和配置才能正常使用。
