# 冠富一键智能抠图工具使用教程

## 一、基础使用步骤

### 1. 访问工具
打开浏览器，输入官方网址：
- 主站：[http://www.guanfu.org](http://www.guanfu.org)
- 抠图专用地址：[https://koutu.netlify.app](https://koutu.netlify.app)

### 2. 上传图片
- **方式一**：点击页面中央的"上传图片"按钮，从本地设备选择需要处理的图片（支持JPG、PNG、WebP等常见格式）
- **方式二**：直接将图片文件从电脑管理器中拖放至网页的上传区域

### 3. 自动抠图
上传完成后，系统会自动启动智能抠图流程，无需手动操作：
- 处理过程中会显示进度提示
- 一般5秒内即可完成抠图（视图片大小和设备性能略有差异）
- 处理完成后会同时展示原图和抠图结果（透明背景）

### 4. 下载结果
点击"下载图片"按钮，获取透明背景的PNG格式抠图结果，可直接用于后续设计或编辑。


## 二、进阶功能使用

### 1. 背景替换
- 抠图完成后，点击"更换背景"选项
- 可选择纯色背景、渐变背景或上传自定义背景图片
- 调整前景位置和大小，预览效果后点击"应用"

### 2. 边缘优化
- 针对毛发、复杂边缘等细节，点击"边缘优化"按钮
- 系统会自动精细化处理边缘像素，提升抠图自然度
- 可手动调整优化强度，满足不同场景需求

### 3. 尺寸调整
- 在预览界面找到"尺寸调整"功能
- 输入目标宽度和高度，或选择预设比例（如1:1、4:3等）
- 保持比例缩放或自由调整，点击"确认"应用


## 三、不同场景使用技巧

### 1. 人像抠图
- **最佳实践**：选择光线充足、背景与人物对比明显的图片
- **优势**：智能识别发丝细节，保留自然过渡效果
- **应用**：制作证件照、头像、人像海报等

### 2. 物体提取
- **适用对象**：产品、动植物、建筑等各类物体
- **技巧**：确保物体轮廓清晰，避免与背景颜色过于接近
- **应用**：电商商品图制作、素材收集、设计合成等

### 3. 多平台适配
- **PC端**：支持鼠标拖拽操作，可精确调整细节
- **移动端**：触摸操作上传图片，适配手机屏幕尺寸
- **PWA应用**：在浏览器中选择"安装"，可离线使用基础功能


## 四、常见问题解决

### 1. 上传失败
- 检查图片格式是否支持（推荐PNG/JPG）
- 确认图片大小不超过5MB（过大图片建议先压缩）
- 刷新页面或更换浏览器重试（推荐Chrome、Edge最新版本）

### 2. 抠图效果不佳
- 尝试更换角度拍摄的图片
- 使用"边缘优化"功能手动调整
- 避免背景与主体颜色过于相似的图片

### 3. 无法下载结果
- 检查浏览器下载权限是否开启
- 确认图片处理已完成（进度条100%）
- 尝试右键点击结果图选择"另存为"


## 五、快捷键与高效操作

| 操作 | 快捷键 |
|------|--------|
| 上传图片 | Ctrl + U（PC）/ Command + U（Mac） |
| 下载结果 | Ctrl + S（PC）/ Command + S（Mac） |
| 重新处理 | Ctrl + R（PC）/ Command + R（Mac） |
| 切换视图（原图/结果） | Tab键 |
