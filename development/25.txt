 System Name : Luhui WHOIS Information System V1.0.4
 系统名称 : 鲁虺域名WHOIS与IP信息查询系统 html JavaScript V1.0.4

    查询无响应，修改下

我对系统进行了优化，解决了查询无响应的问题。主要改进包括增加超时时间控制、优化错误处理机制和添加用户反馈，确保查询过程更加可靠。


这次更新主要解决了查询无响应的问题，主要改进点包括：

1. **增加超时控制机制**：
   - 为每个网络请求设置明确的超时时间（8-10秒）
   - 超时后自动终止请求并尝试备选方案

2. **多服务器和代理备选方案**：
   - 为每个WHOIS查询提供多个备选服务器
   - 集成多个WebSocket代理服务，一个失败自动尝试下一个
   - DNS和地理位置查询也实现了多服务备份机制

3. **实时进度反馈**：
   - 添加了查询进度条，清晰显示当前查询阶段
   - 实时更新进度文本，告知用户当前正在执行的操作
   - 显示当前使用的服务器信息，增加透明度

4. **增强的取消功能**：
   - 添加"取消查询"按钮，允许用户随时终止无响应的查询
   - 使用AbortController确保请求能被彻底终止，释放资源

5. **优化的错误处理**：
   - 更详细的错误信息和可能原因分析
   - 每个步骤出错时提供重试选项
   - 失败时自动尝试其他可用服务，减少用户干预

这些改进大幅提高了系统的可靠性和用户体验，即使某些服务器或代理无响应，系统也能自动切换到备选方案，确保查询能够成功完成。
